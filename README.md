# Pokemon-wiki-Python-app
Just a simple customtkinter app to search up information about any pokemon, it uses https://pokeapi.co/api/v2 to get all its information

To update the app for future generations, just run the "b_everything.py" script then run the "Pokewiki_GUI.py"
